What is EasyCharts?


It's a simple web application that allows you to input data that you can then view as a chart. It makes it easy to views things on the go without the need of creating a new chart via tools such as PowerPoint or Excel.The tool is 100% free to use for any form of graphing you need to do, just simply input the data and the graph corresponds to all user input. Lastly, this tool was mainly made to help NYState employees keep track of necessary data with ease. Feel free to use is as intended and submit any bugs via email at ed.jobe@yahoo.com.



How to use it?

Simply input the intended data inside the input fields clearly specified on each chart page. Once you've placed an input you should see the page dynamically load to incorporate the new chnages otherwise refresh the page and try again. This Application is very lighweight and easy to use, mess around with different charts to visualize how they look. Once you've entered all the values you've like simply hit the download button to get it in excel format. Yep, it's that simple!
